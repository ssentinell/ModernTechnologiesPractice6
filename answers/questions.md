### Ответ 1: Maven
**Maven** - инструмент сборки и управления зависимостями:
- Управление зависимостями (автоматическое скачивание библиотек)
- Стандартная структура проекта
- Жизненный цикл сборки (clean, compile, test, package)
- Декларативная конфигурация через pom.xml
- Репозитории (центральный, локальный, корпоративный)

### Ответ 2: Зависимости
- **Прямая зависимость** - библиотека, явно указанная в pom.xml вашего проекта
- **Транзитивная зависимость** - библиотека, нужная вашей прямой зависимости
  (скачивается автоматически, но не указана явно в pom.xml)

### Ответ 3: PreparedStatement
- Предотвращает SQL Injection через параметризацию запросов
- Предварительная компиляция запроса (лучшее быстродействие при повторном использовании)
- Автоматическое экранирование специальных символов

### Ответ 4: ORM
**ORM (Object-Relational Mapping)** - технология связывания объектов приложения с таблицами БД:
- Решает impedance mismatch (несоответствие между ООП и реляционной моделью)
- Автоматическое преобразование объектов ↔ записи в БД
- Работа с данными через объекты, а не SQL
- Упрощает CRUD операции

### Ответ 5: HQL vs SQL
- **HQL** работает с классами и полями (объектная модель)
- **SQL** работает с таблицами и столбцами (реляционная модель)
- HQL запросы переносимы между разными СУБД
- HQL поддерживает полиморфизм и наследование
