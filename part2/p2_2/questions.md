# Вопросы по JDBC - Часть 2

## Вопрос 1: LIKE с %

**Вопрос:** Почему используется LIKE с % в `findByTitle`?

**Ваш ответ:**  
Оператор `LIKE` используется для поиска по шаблону, а символ `%` означает любое количество любых символов.  
Выражение `"%Матрица%"` ищет фильмы, у которых слово `Матрица` встречается в любом месте строки, а не только как полное точное совпадение.

## Вопрос 2: PreparedStatement vs Statement

**Вопрос:** Что такое PreparedStatement и почему он безопаснее Statement?

**Ваш ответ:**  
`PreparedStatement` — это параметризованный SQL-запрос, который сначала компилируется, а потом получает значения через методы вроде `setString()`, `setInt()` и т.д.  
Он безопаснее `Statement`, потому что не склеивает SQL из строк вручную и тем самым защищает от SQL Injection. Кроме того, он удобнее и часто эффективнее при повторном выполнении одного и того же запроса с разными параметрами.

---

## Короткий вывод

- `LIKE` + `%` нужен для поиска подстроки.
- `PreparedStatement` безопаснее и правильнее, чем обычный `Statement` со строковой конкатенацией.
