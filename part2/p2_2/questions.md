### Ответ 1:
`%` - это wildcard символ в SQL LIKE:
- `%текст` - заканчивается на "текст"
- `текст%` - начинается на "текст"
- `%текст%` - содержит "текст" в любом месте

Используем `%текст%` для поиска подстроки в названии фильма.

### Ответ 2:
**PreparedStatement:**
- Предварительно компилирует SQL запрос
- Использует параметры (?) вместо конкатенации строк
- Защищает от SQL Injection
- Может повторно использоваться с разными параметрами
- Лучшая производительность при многократном выполнении

**SQL Injection пример:**
```java
// Опасно с Statement:
String input = "' OR '1'='1";
stmt.execute("SELECT * FROM users WHERE name = '" + input + "'");
// Выполнится: SELECT * FROM users WHERE name = '' OR '1'='1'
// Вернет все записи!

// Безопасно с PreparedStatement:
pstmt = conn.prepareStatement("SELECT * FROM users WHERE name = ?");
pstmt.setString(1, input);
// Ищет пользователя с именем "' OR '1'='1" (которого нет)
```
